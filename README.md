# 📊 Phone Sales Dashboard (Power BI)

## 🔍 Overview
This project is a **Power BI dashboard** built on a phone sales dataset to analyze sales performance, customer behavior, and product trends.

The dashboard provides insights into:
- Total Sales, Quantity, Transactions, and Average Sales  
- Monthly and daily sales trends  
- City-wise sales distribution (Map visualization)  
- Mobile model and brand performance  
- Payment method usage  
- Customer ratings and demographics  

---

## 📌 Key Insights
- 📈 Sales are relatively stable across months with slight peaks mid-year  
- 🏙️ Cities like Delhi, Mumbai, and Bangalore contribute the most revenue  
- 📱 Apple and Samsung lead in total sales value  
- 💳 Payment methods are almost evenly distributed, with slight preference for UPI  
- ⭐ Majority of customers give ratings between 4 and 5
